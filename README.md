# Credible Impact Communication — MVP

> Workflow-Plattform für glaubwürdige Nachhaltigkeitskommunikation. Erkennung riskanter Claims, Evidenz-Management, Reviews, Freigaben und Exporte.

## Überblick

Dieses Repository enthält eine lauffähige **MVP-Demo** für die Plattform „Credible Impact Communication". Die Anwendung unterstützt Kommunikationsagenturen und Unternehmen dabei, ihre Nachhaltigkeitskommunikation systematisch auf Greenwashing-Risiken zu prüfen, mit Evidenz zu unterlegen und über strukturierte Freigabeprozesse abzusichern.

Das Frontend-Design orientiert sich am visuellen Erscheinungsbild von [Grüne Welle Kommunikation](https://gruenewellepr.de/).

## Technologie

| Komponente | Technologie |
|---|---|
| **Framework** | Next.js 15 / React 19 |
| **Sprache** | TypeScript |
| **Hosting** | Vercel |
| **Daten** | Eingebettete Demo-Daten |

## Funktionsbereiche

| Bereich | Beschreibung |
|---|---|
| **Überblick** | KPI-Metriken, prioritäre Projekte, Review-Warteschlange |
| **Projekte** | Projektübersicht mit Risikobewertung und Freigabe-Status |
| **Asset-Prüfung** | Inline-Claim-Review mit Evidenz-Kontext und Formulierungsvorschlägen |
| **Evidenz-Hub** | Strukturierte Nachweise mit Gültigkeit und Verknüpfung |
| **Freigaben** | Review- und Approval-Queue mit Status-Tracking |
| **Berichte** | Exportierte Audit-Ergebnisse (PDF, CSV) |
